Fullstack Academy Capstone Project
Faker Blues O-Ring Book App

In wireline operations, efficient and consistent maintenance on downhole instruments is crucial to completing necessary oil and gas well intervention. Tool maintenance manuals are large, become damaged and contain far more information than required for day to day operations. To eliminate this bulk and keep the required specs on hand, I am developing the Faker Blues O-Ring Book app.

The app will allow users to browse tools by name or series number and see vital info such as O-ring size, connection type, length, weight, and operating voltage. Regular users will be able to log in and see all tools and related data and existing tool strings. Admin users will be able to log in, add or edit tools and tool strings.
